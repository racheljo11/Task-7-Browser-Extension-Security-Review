# Task-7-Browser-Extension-Security-Review
Browser extension security check and verification
Objective

To review installed browser extensions, identify any potentially harmful or suspicious extensions, and remove them if found.

 Browser Used

Google Chrome

 Review Process

The following steps were performed:

Opened chrome://extensions/

Reviewed all installed extensions

Checked:

Developer information

User reviews and ratings

Permissions requested

Installation source

Researched extensions online to verify legitimacy

Checked for unusual browser behavior (pop-ups, redirects, ads)

The extension Google Docs Offline was researched online and confirmed to be an official extension developed by Google. It is safe and used to enable offline document editing.

Suspicious Extensions Found

 Security Best Practices Followed

Verified extensions were installed from Chrome Web Store

Checked permissions for excessive data access

Ensured browser is updated

Reviewed extensions for unusual behavior

Removed unused extensions (if any)

 Conclusion

Browser extensions can pose security risks if not regularly reviewed. After careful analysis and verification, no unsafe extensions were found. This indicates a secure and properly maintained browser environment.

Regular extension audits are recommended to prevent malware, spyware, and data theft.
